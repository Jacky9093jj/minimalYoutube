# Minimal YouTube

Minimal YouTube is a browser extension that hides distracting parts of YouTube to give you a cleaner viewing experience.

## Features

- Hides Shorts
- Hides related videos
- Hides comments
- Hides live chat
- Hides home feed

## How it works

The extension automatically hides selected YouTube elements using CSS injection and a page observer.

## Installation

1. Download or clone this repository
```bash 
git clone https://github.com/Jacky9093jj/minimalYoutube.git
```
2. Open your browser’s extension page
3. Enable Developer Mode
4. Load the extension folder as an unpacked extension
